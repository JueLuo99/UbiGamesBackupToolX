# UbiGamesBackupToolX

![](https://i.loli.net/2020/07/25/G5pnR3w9cWxeBMK.png)

## 适用群体
- 需要备份不支持 Uplay 云存档的单机游戏的玩家（如：《刺客信条：黑旗》《细胞分裂：黑名单》等）
- 无法正常通过 Uplay 云存档备份存档的玩家

## 现有特性
- 自动探测存档路径
- 支持检测多个 Uplay 用户
- 还原存档
- 开机自动启动
- 实时备份游戏（暂时仅支持部分游戏）
- “反选”功能
- “搜索”功能

## 计划更新
- 接入第三方云以储存备份
- 能够备份一些并不在Uplay安装目录下游戏的备份（例如：《刺客信条：兄弟会》等）
- 快捷键的支持
